Recipe Help Saving a Recipe
To save a recipe, press the button in the header. The following information needs to be provided:
  • Recipe Title
  • Recipe Description
  • A picture
  • Serving size
  • Preparation and cooking time*
  • Atleast 1 listed ingredient
  • Atleast 1 listed instruction
If any of these values are missing while attempting to save the recipe, a helpful message will appear onscreen displaying what the error is.

*Value is based on time format, so 60 = 60minutes. The value is divided in hours and minutes. Values over 59 will be automatically converted to the appropriate time format when saved.
Adding ingredients and instructions An ingredient needs an amount, unit and name. select 'N/A' as unit if you want this to stay empty. After filling in the information in the ingredient / instruction field, press the button to list the item. Edit recipes / instructions Press the button. This will activate Edit mode. Your ingredients or Instructions are now editable. Press again to disable edit mode.

Note: When activating Edit mode on Instructions, the Reordering mode will be disabled. Delete recipes / instructions Swipe right on the desired ingredient / instruction. The button will appear on the left. Press the button to delete the entry. Reordering instructions Press the button to activate Reordering mode. While this mode is active, you can now drag and reorder ingredients. Activating reorder mode deactivates the swipe to delete. Deactivate the mode by pressing the button again to be able to delete the entry.
